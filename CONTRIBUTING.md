# Contributing

Thank you for improving the project.

- Keep the feed minimal and attributed; do not add user tracking or a historical event archive.
- Preserve the public schema or document a versioned migration.
- Add or update tests for collector changes.
- Test markup in TRMNL's preview before proposing visual changes.
- Do not submit venue/event data obtained from private, authenticated, or prohibited sources.
