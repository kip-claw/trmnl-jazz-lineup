# Roadmap

## Before public Recipe submission

- [ ] Capture and review a real 800×480 TRMNL render.
- [ ] Receive or document Jazz Lineup's response to the limited, attributed reuse request.
- [ ] Tag the first stable Recipe release after the above checks.

## Afterwards

- [ ] Add a separate small-display Recipe, rather than shrinking the complete board.
- [ ] Add a documented visual-regression workflow once a stable renderer/fixture is available.
- [ ] Consider additional display preferences only when the source data supports them reliably.

This is deliberately not a plan to mirror or archive Jazz Lineup's calendar.
